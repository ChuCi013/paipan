使用说明

排盘参考：《增删卜易》《卜筮正宗》等；梅花易数算法取自《梅花易数》等；六十四卦卦名/卦辞/爻辞参考《周易》（即《易经》）等。

  • 顶部切换"六爻"或"梅花"模式，万年历/干支区域共用。
  • 六爻模式：输入六个数字(0-3)排盘。0老阴(变阳) 1少阳 2少阴 3老阳(变阴)。
  • 六爻排盘自动排纳甲干支、六亲、六神、世应、旬空、飞伏神。
  • 万年历：公历日期自动推算干支（年干支立春分界，月干支节气近似），均可手动修改。
  • 日干支影响六神排列、旬空、日辰标记；月干支影响月建标记；时干支仅供记录。
  • 动爻红色"动"，月建黄色"月"，日辰蓝色"日"，旬空灰色删除线。
  • 日冲旬空之爻标记为"冲空动"（暗动），日冲旺动之爻为"日破"。
  • 点击六爻表中的爻行，弹出该爻详细分析（化进化退、月日动影响、回头生克合绊）。
  • 点击卦名可查看该卦卦辞与爻辞；点击卦盘爻象可查看对应八卦类象。
  • "一键复制"可将排盘结果整理为纯文本，可粘贴给你会卜算的朋友进行卦象解读。
  • 梅花模式：输入三个正整数起卦。数1÷8余为上卦，数2÷8余为下卦，数3÷6余为动爻。算法取自《梅花易数》。
  • 梅花显示本卦、互卦、变卦三卦并排，含体用关系。
  • 干支显示区下方显示十神简要解释（以日干为中心）。

零基础入门指南

🔮 六爻怎么用（需要硬币）

不疑不占 · 不诚不占 · 一事一占

第0步：打开页面后，先点「今日」按钮，自动填好今天的日期和干支（排盘需要用到）。

第1步：准备三个一样的硬币。

把纹理复杂的那一面当作反面（比如有花、有字的那面），纹理简单的那一面当作正面。

第2步：双手合住三个硬币，心里默念你想问的事，在手心里摇晃，然后松开让硬币自然落下。

看结果，数一下几个正面、几个反面：

🔵 0个正面（全是反面） → 输入 0（老阴，会变阳）
🔵 1个正面 + 2个反面 → 输入 1（少阳，不变）
🟡 2个正面 + 1个反面 → 输入 2（少阴，不变）
🟡 3个正面（全是正面） → 输入 3（老阳，会变阴）

第3步：把数字填入第1个空位（页面上有6个空位）。重复第2步，一共掷6次，按顺序把6个空位填满。

⚠️ 第一次是"最下面那一爻"（初爻），第六次是"最上面那一爻"（上爻）。不知道啥是初爻上爻？没关系，按次数顺序填就对了。

第4步：填完后点「排盘」按钮，下面就会出现完整的排盘结果——卦名、世应、六亲、六神全自动给你排好。

第5步：点「📋 复制」，粘贴给你会卜算的朋友解读。

🌸 梅花易数怎么用（不需要道具）

不疑不占 · 不诚不占 · 一事一占

第1步：脑子里想好三个正整数。可以是日期、时间、看到的数字、心里冒出来的数——什么都行。

比如：你看了下表是10:25，就用10、2、5（把25拆成2和5也行）。或者随手翻书第37页，用3、7、37。

第2步：切到「🌸 梅花」模式，把三个数填进去。

第一个数 → 上卦 | 第二个数 → 下卦 | 第三个数 → 动爻

第3步：点「排盘」，本卦、互卦、变卦一起出来，还有体用生克关系。

第4步：点「📋 复制」，粘贴给你会卜算的朋友解读。
